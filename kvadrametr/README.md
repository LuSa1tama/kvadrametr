# КвадраМетр — Прогнозирование цен на недвижимость

## Описание задачи

Задача регрессии для предсказания рыночной стоимости квартиры. На вход модели подаются площадь, район, этаж, год постройки, расстояние до метро и инфраструктура. Целевая переменная — цена в рублях. Используются только классические алгоритмы машинного обучения без глубокого обучения.

## Данные

Датасет из 2000 объектов в формате CSV. Данные сгенерированы программно на основе статистических распределений рынка недвижимости России (ЦИАН, Яндекс.Недвижимость, Росстат 2023–2024). Файл создаётся автоматически при запуске `create_data.py`.

## Методы машинного обучения

Реализованы и сравнены шесть алгоритмов:

- Linear Regression
- Ridge Regression
- Decision Tree
- K-Nearest Neighbors
- Random Forest
- Gradient Boosting

Предобработка включает кодирование районов (`LabelEncoder`) и нормализацию числовых признаков (`StandardScaler`). Выбор оптимальной модели по метрикам R² и MAPE. Обученная модель сохраняется в `models/model.pkl`.

## Веб-приложение

Реализовано на Flask. Форма ввода параметров квартиры → прогноз цены с диапазоном и сравнительной таблицей всех моделей.

## Запуск проекта

1. Установка зависимостей:

```
pip install -r requirements.txt
```

2. Генерация данных и обучение модели:

```
python create_data.py
```

3. Запуск веб-сервиса:

```
python app.py
```

Интерфейс доступен по адресу: http://127.0.0.1:5000

4. Анализ данных — ноутбук:

```
notebooks/real_estate_analysis.ipynb
```
